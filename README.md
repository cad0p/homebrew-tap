# Cad0p Tap

## How do I install these formulae?

`brew install cad0p/tap/<formula>`

Or `brew tap cad0p/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cad0p/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
